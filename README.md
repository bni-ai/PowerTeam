# 華AI PowerTeam

協助 BNI 協會的成員，打造屬於他們自己的 Power Team。

## 什麼是華AI PowerTeam？

傳統 BNI Power Team 鎖定單一產業鏈，成員按上下游分工。  
華AI 不同——我們的專業是「AI／系統／數位轉型」，橫切所有產業。

**SOP 架構不變，對接內容隨產業調整。**

## Repo 結構

```
PowerTeam/
├── docs/              ← 給人看的（PT 組建手冊、操作指南、盲點分析）
├── knowledge/         ← 給 AI 看的（結構化資料，frontmatter + 規則）
│   ├── sop/           ← 服務交付 SOP
│   ├── roles/         ← 角色定義與分工
│   ├── templates/     ← 交接文件模板
│   └── industries/    ← 產業模板（美業、製造業...）
├── skills/            ← AI 工具（讀 knowledge/ 執行任務）
└── members/           ← 會員資料
```

## 服務交付流程

```
BD 找到客戶 → PM 盤點痛點 → 講師針對性教學 → 顧問規劃方案 → 工程快贏 → 大型建置
                              ↑ 補助申請同步啟動（雙軌並行）
```

## 核心原則

- **先診再教**：PM 先盤點痛點，講師再針對性地教
- **課程與系統拆開**：教課歸教課，系統歸系統
- **4-5 人核心小隊**：深度綁定，不深度擴張
- **客戶分級（ABCD）**：D 黃金客戶優先，C 客戶不接
- **補助是加速器不是引擎**：沒補助也能活

## 分階段交付

| Phase | 產出 |
|-------|------|

| 2 | 對外簡報／頁面 |
| 3 | 活系統（自動配對 + 追蹤引薦） |
