# Power Team 三種類型

## 上中下游關係

| 類型 | 位置 | 定義 | 白話 |
|------|------|------|------|
| 營銷 PT | 上游 | 幫你公司得到更多業務 | 找客戶進來 |
| 營運 PT | 中間 | 幫你公司營運更輕鬆 | 把事做好、交付 |
| 學習 PT | 下游 | 讓你更理解產業趨勢 | 教會、傳承、提升能力 |

## 華AI 的特殊性

傳統分會：每人產業不同，不重疊 → 按產業鏈組 PT。
華AI：多人能力重疊（都會教學、都會架站、都會 AI）→ 不能按產業別分，要按「服務類型」分。

## 一人多 PT、多位置

同一個人在不同人的 PT 裡可以站不同位置：

- 老魚對佑禎 → 營銷 PT（上游）：教 AI 給設計師/房仲 → 引薦給佑禎
- 老魚對客戶 → 營運 PT（中間）：顧問診斷 → 出報告 → 交給工程
- 老魚對成員 → 學習 PT（下游）：教 BNI 成員用 AI

## 兩種運作模式

### 模式 A：產業鏈

客戶有需求 → 順著鏈走：顧問諮詢 → 講師教學 → 員工培訓 → 系統導入 → 客製產品。
每個段落由擅長的人做，做完交給下一段。

### 模式 B：講師聯盟

全員都是講師，各有主題 → 線上課程 / 免費分享會 / 實體交流會 / 企業內訓。
輪流講、聯合接案 → 企業聽完有具體需求 → 切換到模式 A 產業鏈做對接。

### 模式關係

模式 B（講師聯盟）= 入口漏斗，負責獲客。
模式 A（產業鏈）= 交付引擎，負責做事。
前後銜接，不是二選一。
