## 一、每週例會標準流程（約 90 分鐘）

BNI 例會議程在全球 70+ 個國家幾乎完全一致，通常清晨 7:00 開始。

| 環節 | 時間 | 目的 |
|------|------|------|
| 會前交流 | 15-30 分鐘 | 非正式交流、接待訪客 |
| 正式開場與使命宣讀 | 5 分鐘 | 宣示核心價值「付出者收穫」 |
| 教育簡報 | 3-5 分鐘 | 分享社交技巧或 BNI 知識 |
| 分會導師/新會員介紹 | 5 分鐘 | 歡迎新會員、表揚完成 MSP 者 |
| 會員 60 秒商業簡報 | 30-45 分鐘 | 每人介紹本週理想引薦對象（核心環節） |
| 訪客介紹 | 5-10 分鐘 | 訪客自我介紹 |
| 本週專題演講（10 分鐘） | 10 分鐘 | 輪流由一名會員深度展示專業 |
| 引薦與 TYFCB 回報 | 15-20 分鐘 | 傳遞引薦單、回報成交金額 |
| 訪客座談與閉幕 | 5 分鐘 | 訪客入會說明、非正式後續交流 |

**設計邏輯：VCP 原則**
- Visibility（能見度） → Credibility（信任度） → Profitability（獲利度）
- 每週例會的重複出現，持續累積 V 和 C，最終轉換為 P

## 二、九長（Leadership Team）職位與職責

| 職位 | 職責摘要 |
|------|---------|
| **主席 (President)** | 主持例會、激勵會員、召開月度 LT 會議 |
| **副主席 (Vice President)** | 追蹤績效指標、領導「會員委員會」審核新會員 |
| **財務秘書 (Secretary/Treasurer)** | 管理財務、會議計時、追蹤會員續費 |
| **教育協調員 (Education Coordinator)** | 每週教育環節、提升引薦轉化率 |
| **訪客主辦人 (Visitor Host)** | 迎接訪客、會後入會說明、訪客追蹤 |
| **導師協調員 (Membership Coordinator)** | 指派導師、引導新會員完成護照計畫 |
| **活動協調員 (Event Coordinator)** | 社交活動、培訓日、跨分會聯誼 |
| **公關協調員 (PR Coordinator)** | 社群媒體行銷、吸引訪客與新行業加入 |
| **引薦獎勵協調員 (Referral Coordinator)** | 監督引薦品質、鼓勵高品質 1-2-1 |

> 所有職位為義務職，每半年至一年輪替，體現「領導就是服務」精神。

## 三、新會員入會流程

```
受邀訪客 → 會後面談 → 提交申請 → 委員會審核 → 繳費核准
    ↓
正式宣布入會 → MSP 培訓（30天內完成）→ 護照計畫（90天）
```

| 步驟 | 說明 |
|------|------|
| 受邀訪客 | 由現有會員邀請，觀察分會氣氛 |
| 會後面談 | 訪客主辦人說明運作方式、費用、行業排他性 |
| 提交申請 | 填寫申請表、三位專業推薦人 |
| 委員會審核 | 檢查行業是否已有人佔據、背景信譽 |
| 繳費核准 | 入會費 + 年費 |
| MSP 培訓 | 必修，未完成不得進行 10 分鐘專題 |
| 護照計畫 | 90 天內與所有核心產業會員完成 1-2-1 |

## 四、一對一（1-2-1）會面標準格式

**頻率：** 每週 1-2 次，每次 60-90 分鐘
**場地：** 輪流造訪彼此辦公室（實地觀察專業度）
**目標：** 找出「我該如何幫你找到客戶？」，非銷售

### GAINS 表（會面前雙方交換）

| 字母 | 代表 | 說明 |
|------|------|------|
| **G** | Goals 目標 | 未來 6-12 個月的商業或個人目標 |
| **A** | Accomplishments 成就 | 引以為傲的專業成就或認證 |
| **I** | Interests 興趣 | 愛好、情感連結點 |
| **N** | Networks 人脈 | BNI 以外參加的組織（扶輪社、公會等）|
| **S** | Skills 技能 | 核心專業強項 |

### 最佳實踐
- 會面前先研究對方網站，帶著「我想介紹誰給你」的心態
- 結束後 24 小時內在 BNI Connect App 登記並發送感謝訊息
- 核心問句：「遇到什麼樣的人說什麼樣的話，就是你的理想引薦信號？」
