# 03 — 怎麼組 Power Team

## 兩條路找到夥伴

### 路線 A：從引薦找（由上往下）

適合主席、副主席、資訊長用來幫全會配對。

**步驟：**

1. **蒐集**：請每位成員列出三層引薦
   - 一般引薦：最常碰到的客戶類型
   - 理想引薦：最想要的客戶類型
   - 夢幻引薦：做夢都想接到的客戶
2. **分群**：把所有人的引薦攤開，粗分成 8-15 種客群樣貌
3. **找重疊**：哪些人的客群有交集？
4. **配對**：客群重疊的人，安排一對一聊聊
5. **確認**：一對一之後，雙方覺得可以合作 → 組成 PT

```
全員三層引薦
    │
    ▼
 分成 8-15 種客群
    │
    ▼
 找重疊 → 配對一對一
    │
    ▼
 確認合作 → 組成 PT
```

### 路線 B：從自己找（由下往上）

適合每位成員自己主動組隊。

**步驟：**

1. 做完第 2 章的「自我探索四問」
2. 看看你的上游和下游分別需要什麼角色
3. 在分會裡找符合的人
4. 約一對一，確認彼此客群是否重疊
5. 重疊 → 組成 PT

## 組 PT 的三個原則

### 原則一：切片化

一個 PT 只做一件明確的事。不要包山包海。

```
❌ 一個 PT 包：顧問 + 教學 + 系統開發 + 廣告投放 + 全部
   → 工時長、交付慢、溝通成本高

✅ 切成小 PT：
   PT-顧問組 → 診斷完 → 收錢 → 轉介給講師組
   PT-講師組 → 教完 → 收錢 → 轉介給工程組
   PT-工程組 → 做完 → 收錢
```

每個 PT 快交付、快結案、快收錢。
客戶滿意，再轉介給下一個 PT。

### 原則二：一個 PT 不超過 5 人

人太多 → 溝通成本暴增、分工不清。
4-5 人剛好：有上游、有下游、有人做事、有人支援。

### 原則三：做起來要輕鬆

如果一個 PT 裡你做的事讓你覺得很累、很不想做，
那這個位置不是你的 → 換位置或換 PT。

## 你可以同時在多個 PT

一個人可以同時參與多個 PT：

```
PT-美業顧問組 → 你當顧問（營運位）
PT-講師聯盟  → 你當講師（學習位）
PT-佑禎旅宿鏈 → 你當上游引薦（營銷位）
```

只要每個 PT 裡你做的事不會互相衝突、時間上排得開，就可以。

## 全會大約 5-6 個 PT

以華AI 35 位成員來說，不需要 35 個 PT。
大約 5-6 個就夠：

- 按服務類型分（顧問組、講師組、工程組、行銷組）
- 按特定客群分（美業組、中小企業組）
- 按模式分（講師聯盟、產業鏈）

具體怎麼分，看實際配對結果。先從 2-3 個開始試，跑通再擴。
