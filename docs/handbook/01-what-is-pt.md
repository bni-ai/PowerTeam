# 01 — Power Team 三種類型

## 什麼是 Power Team？

Power Team 是一群「服務同類客群」的成員，組成的協力小隊。
你們不是競爭對手，而是同一個客戶在不同階段需要的人。

## 三種類型

Power Team 分成三種，對應上、中、下游：

```
營銷 PT          營運 PT           學習 PT
（上游）    →   （中間）     →   （下游）
帶客戶進來       幫客戶把事做好     讓客戶持續成長
```

### 營銷 PT — 幫你得到更多業務

做什麼：幫你找到客戶、帶來業務機會。
白話：「有人幫我把客戶送到門口。」

例子：
- 你是室內設計師，有人專門辦建築展覽，參展的業主就是你的潛在客戶
- 你是 AI 顧問，有人開企業主社群，裡面的老闆就是你的潛在客戶

### 營運 PT — 幫你公司跑得更輕鬆

做什麼：讓你的公司營運更省力，上下游補齊你缺的環節。
白話：「有人幫我把後面的事扛起來，我專心做我擅長的。」

例子：
- 你接了顧問案，診斷完需要有人做系統 → 工程夥伴幫你做
- 你教完課，客戶想導入 AI → 導入顧問幫你接手

### 學習 PT — 讓你跟上趨勢

做什麼：讓你更理解產業動態、提升專業能力。
白話：「有人陪我一起學，我不會落後。」

例子：
- 一群講師組讀書會，每月分享各自領域的新知
- 幾個做 AI 的人固定交流，互相測試新工具

## 上中下游怎麼接？

一個完整的服務鏈，三種 PT 串在一起：

```
營銷 PT 找到客戶
    │
    ▼
營運 PT 把事做好（診斷→規劃→交付）
    │
    ▼
學習 PT 教會客戶（教學→培訓→持續成長）
```

不是每個案子都走完整條鏈。
有的客戶只需要「教學」，有的只需要「做系統」。
按需求切，不硬湊。

## 華AI 的特別之處

傳統分會：每人產業不同（水電、律師、保險），天生不重疊。
華AI：很多人能力重疊（都會教學、都會架站、都會用 AI）。

所以我們不按「產業別」組 PT，而是按「服務類型」組：

| 服務類型 | 做什麼 |
|---------|--------|
| 顧問類 | 診斷、規劃、諮詢 |
| 講師類 | 教學、內訓、課程 |
| 工程類 | 系統開發、AI 導入、自動化 |
| 行銷類 | 廣告投放、品牌、內容製作 |

你擅長哪個，就站哪個。下一章教你怎麼找到自己的位置。
