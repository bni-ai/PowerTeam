# Power Team 使用手冊

## 這本手冊是什麼？

這是華AI分會的 Power Team（協力團隊）組建指南。
讀完之後，你會知道：

- Power Team 有哪三種類型
- 你適合站在哪個位置
- 怎麼找到跟你客群重疊的夥伴
- 怎麼跟夥伴分工、交接、收錢

## 給誰看？

華AI分會的每一位成員。
不管你是剛加入還是已經待了一年，都適用。

## 怎麼用？

**快速入門**：讀第 1～3 章，就能自己動手組 PT。
**進階理解**：第 4～6 章講運作模式、交接流程和實際案例。

| 章 | 標題 | 你會知道 |
|----|------|---------|
| 01 | PT 三種類型 | 營銷、營運、學習分別是什麼 |
| 02 | 找到你的位置 | 我最擅長什麼、適合站哪裡 |
| 03 | 怎麼組 PT | 怎麼找夥伴、一個 PT 該多大 |
| 04 | 兩種運作模式 | 產業鏈接力 vs 講師聯盟獲客 |
| 05 | 交接流程 | 做完交什麼、怎麼轉介 |
| 06 | 案例 | 看別人怎麼組的 |

## 重要觀念

這本手冊是**框架**，不是規定。
沒有人會指定你站哪個位置——你自己選最輕鬆的那個。
不同案子、不同夥伴，你可以換位置。
